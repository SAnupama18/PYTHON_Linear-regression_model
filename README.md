CSV Data Analysis using Pandas and Matplotlib

Project Overview

This project demonstrates how to load a CSV file using Pandas, perform basic data analysis, and visualize data using Matplotlib and Seaborn.

Features

- Load CSV data
- Display dataset information
- Calculate average marks
- Find highest and lowest scoring students
- Generate Bar Chart
- Generate Scatter Plot
- Generate Heatmap

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

Project Files

- analysis.py
- data.csv
- README.md

How to Run

1. Install required libraries:

pip install pandas numpy matplotlib seaborn

2. Run the project:

python analysis.py

Output

- Dataset summary
- Average marks
- Highest and lowest scoring students
- Bar chart visualization
- Scatter plot visualization
- Correlation heatmap
