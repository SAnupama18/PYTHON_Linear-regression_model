## House Price Prediction

## Project Overview
This project uses Linear Regression to predict house prices using housing data.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Dataset
The dataset contains housing features and prices.

## Features
- Data preprocessing
- Train-test split
- Linear Regression model
- Price prediction
- Data visualization

## Results
- R² Score: 0.6688
- Mean Squared Error: 24.2911

## How to Run

1. Install required libraries:
   pip install pandas numpy matplotlib scikit-learn

2. Run the project:
   python house_price.py
